# sholdee Homebrew Tap

Homebrew formulae for tools maintained by sholdee.

## drydock

Install:

```bash
brew install sholdee/tap/drydock
```

Or tap first:

```bash
brew tap sholdee/tap
brew install drydock
```

drydock releases and formula updates are managed from
[sholdee/drydock](https://github.com/sholdee/drydock).
